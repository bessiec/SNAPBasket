Overview

Unify is an incredibly beautiful responsive Bootstrap Template for corporate and creative professionals. It works on all main web browsers, tablets and phones. Current upgraded release is v1.3 and much more new amazing features are coming soon in the next releases. Thank you for using Unify.

Main Structures top

Template files consist of top, header, slider, content part, footer and copyright

Beginning of Page

Below code is used at the beginning of all HTML pages to detect Internet Explorer browser version and set a spesific class applied to Internet Explorer versions.

<!DOCTYPE html>
<!--[if IE 8]> <html lang="en" class="ie8"> <![endif]-->
<!--[if IE 9]> <html lang="en" class="ie9"> <![endif]-->
<!--[if !IE]><!--> <html lang="en"> <!--<![endif]-->

Header & Navigation top

Unify Template consist in two basic navigations (menu) and you can use any of them in your project. Here has shown how you can connect navigations.

If you want to use an alternative header, you should change in head the style's name:

<head>
    <!-- This is default header style -->
    <link rel="stylesheet" href="assets/css/headers/header1.css" />
    
    <!-- OR (This is an alternative header style) -->
    <link rel="stylesheet" href="assets/css/headers/header2.css" />
</head>

The default header style is assets/css/headers/header1.css


Moreover, do not forget to connect top and header as shown below:

<!--=== Top ===-->
<div class="top">
    <div class="container">
	    ...
    </div><!--/container-->
</div><!--/top-->
                
<!--=== Header ===-->
<div class="header">
    <div class="container">
	    ...
	    ...
    </div><!--/container-->
</div><!--/header-->

Make sure you have changed top and header blocks for alternative header as they have different codes inside blocks.

Theme Configuration top

All theme css files located in assets/css/themes/..

To change theme color you should just replace assets/css/themes/default.css to assets/css/themes/your_theme.css or you can use ready theme css files such as:

  • assets/css/themes/red.css and assets/css/themes/header1-red.css
  • assets/css/themes/blue.css and assets/css/themes/header1-blue.css
  • assets/css/themes/light.css and assets/css/themes/header1-light.css

v1.2 Upgrade

Here is breifly explained what we have changed in v1.2:

  • File structures:
    • Changed HTML file names. For example:
      • faq.html to page_faq.html
      • about.html to page_about.html
      • contact.html to page_contact.html
      • icons.html to feature_icons.html
      • table.html to feature_table
      • and others...
    • We added all plugins to plugin folder and now all plugins located in assets/plugins/..
  • Parallax Slider now on autoplay. Modified file: assets/plugins/parallax-slider/js/jquery.cslider.js
  • Changed folder and file names from menu to header in CSS folder assets/css/headers/header1.css and assets/css/headers/header2.css
  • Changed Modernizr JS file name assets/js/modernizr.custom.js
  • Changed top and header ID names to CLASS names and now they are <div class="top"> and <div class="header">
  • Changed Social Icons class names which is in footer and in other pages.
  • Deleted tag <div id="body"> in all HTML files
  • Added favicon icon <link rel="shortcut icon" href="favicon.ico">
